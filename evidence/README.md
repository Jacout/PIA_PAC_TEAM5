Evidence requirements
=====================

Colocar en esta carpeta las siguientes evidencias (nombres exactos):

- `run_hash.png`: captura de pantalla del comando `./bin/pia_team5_debug hash ...` mostrando salida y `pia_team5.log`.
- `run_sniff.png`: captura de la ejecución `sudo ./bin/pia_team5_debug sniff lo` y/o captura de Wireshark/tcpdump. Asegúrese de usar red de laboratorio/loopback.
- `static_analysis.png`: captura de Ghidra o Radare2 mostrando funciones y strings relevantes.
- `demo_video.mp4`: video de 2–5 minutos demostrando las funciones principales (hash, sniff, processes).

Además incluir:
- `logs.txt` (opcional): archivo de log consolidados.

Notas de seguridad: realizar todos los ensayos en entornos controlados. No capture tráfico de terceros ni use interfaces de producción.
